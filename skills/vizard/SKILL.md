# Vizard MCP Skill

Use this skill when:
- user wants TikTok clips
- user wants YouTube Shorts
- user wants Instagram Reels
- user wants viral podcast highlights
- user wants AI subtitle editing
- user wangts social publishing

This skill requires the Vizard MCP server.

## Available Tools

- create_clips
- create_clips_advanced
- edit_short_video
- wait_for_project_completion
- query_project
- list_social_accounts
- create_clips_and_wait
- generate_social_caption
- publish_video
- viral_pipeline

## Example Prompts

- Turn this YouTube video into TikTok clips
- Generate viral shorts from this podcast
- Create AI captions for this short video
